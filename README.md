# API Lab Assignment

This project uses the PokéAPI with Python.

The purpose of this project is to practice working with an API, making HTTP requests, reading JSON responses, and printing clean output to the terminal.

## API Used

PokéAPI  
https://pokeapi.co/

## What the Program Does

The program makes 3 different API calls:

1. Gets information about Pikachu
2. Gets information about the Electric type
3. Gets information about the move Thunderbolt
